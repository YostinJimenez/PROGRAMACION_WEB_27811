About

Freelancer es un tema de Bootstrap gratuito y de uso, creado por Start Bootstrap. La descarga incluye los archivos fuente completos, incluyendo HTML, CSS y JavaScript, así como hojas de estilo SASS opcionales para una fácil personalización.

Puedes crear tu propio avatar personalizado para el encabezado, cambiar el icono en los divisores y agregar tu dirección de correo electrónico al formulario de contacto para que sea completamente funcional.
